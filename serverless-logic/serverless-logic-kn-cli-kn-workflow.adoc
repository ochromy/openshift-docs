:context: serverless-logic-kn-cli-kn-workflow

{ServerlessLogicProductName} includes a plugin named `kn-workflow` for Knative CLI, which provides command line features that help you develop a local workflow projects quickly.

This document describes how you can install and use the kn-workflow plug-in in OpenShift Serverless Logic.

You can also find brief introduction for some of the commands that the plugin provides.
